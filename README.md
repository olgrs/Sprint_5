# Sprint_5 — UI-автотесты для учебного сервиса «Доска»

Проект с автотестами на Python + Selenium для проверки функциональности сервиса «Доска» (https://qa-desk.education-services.ru/).  
Разработан в рамках курса Яндекс Практикум, спринт «UI-тестирование».

## Тестовые сценарии

Покрыты следующие функциональности:

- **Регистрация пользователя**  
  - Успешная регистрация нового пользователя  
  - Регистрация с email не по маске (негативный)  
  - Регистрация уже существующего пользователя (негативный)
- **Авторизация (Login)**  
  - Успешный вход зарегистрированного пользователя
- **Выход из системы (Logout)**  
  - Разлогинивание авторизованного пользователя
- **Создание объявления**  
  - Попытка создания без авторизации (проверка модального окна)  
  - Создание объявления авторизованным пользователем (заполнение всех полей, выбор категории/города/состояния, публикация и проверка в профиле)

## Запуск тестов

1. Установите зависимости:
    pip install selenium pytest
2. Убедитесь, что установлен браузер Google Chrome и соответствующий ChromeDriver (Selenium 4 автоматически управляет драйвером при наличии браузера).
3. Запустите все тесты из корня проекта:
    pytest tests/
Или конкретный файл:
    pytest tests/test_registration.py

## Требования
